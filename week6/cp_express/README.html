Notes Regarding Rubric Items:

Carousel

The sequence of images is automated and can't be controlled by the user. I wanted a vanilla javascript solution that allowed for a fading transition, but couldn't find one, so I hacked this together using CSS keyframe animation.

It needs to be event-driven, but I spent so much time getting it to work using sequential logic that a revised version wasn't in the cards.

Scrollbar

The scrollbox is on the "Birds and the Biosphere" page. It allows the user to see all of the immense image that it obscures. I realize that, in practice, an image of this size (on disk) would be too large for a responsive site.

Form Submission

This is done using the Nodemailer library and an SMTP relay service (SendGrid). Email goes to a Gmail account that is setup on a registered domain (charmingplumage.com). Initially, I had trouble getting the SMTP server results in the rendered page, but then I realized that I could just the Express Handlebars render() function as the callback to the sendMail() function and pass in the info/error objects that way.
